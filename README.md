## How to run?

1. Create a virtual environment:

```bash
 conda create -n jarvis python=3.11 -y
```
 
2. Activate virtual environment:

```bash
 conda activate jarvis
```


3. Install requird packages:

```bash
 pip install -r requirements.txt
```

4. Run the JARVIS script:

```bash
 python jarvis.py
```
